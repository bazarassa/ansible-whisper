# Role for whisper.cpp

## Installation

At first, install the ansible itself
`sudo apt install ansible` or `sudo dnf install ansible`

After that, run a playbook:
For example, for localhost `ansible-playbook playbook.yaml -K -i localhost`. You can add any inventory you want.

For whisper.cpp documentation, see https://github.com/ggml-org/whisper.cpp/blob/master/README.md
